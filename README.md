# Fabric Agent Downloads

Official distribution repository for Fabric Agent.

## Available Downloads

### macOS

- **Apple Silicon** (M1, M2, M3, M4): macOS 12.0+
- **Intel** (x86_64): macOS 12.0+

### Installation

1. Download the appropriate version for your Mac
2. Extract and move `FabricAgent.app` to `/Applications`
3. Right-click and select "Open" on first launch
4. Follow enrollment instructions

### Verification

Verify downloads with SHA256 checksums provided in release files.
